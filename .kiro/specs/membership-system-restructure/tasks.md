## Notes

- Tasks marked with `*` are optional and can be skipped for a faster MVP
- Each stage ends with a PHPUnit checkpoint â€” never proceed to the next stage with a failing test suite
- The GoCardless webhook endpoint (`POST /gocardless/webhook`) must remain CSRF-exempt throughout â€” verify `VerifyCsrfToken` exclusion is intact after any middleware changes
- `composer.json` autoload (`"BB\\": "app/"`) requires no changes â€” all namespace renames are covered by the PSR-4 prefix
- Property tests (6.17, 6.18, 5.3) act as regression guards and can be run at any point after their stage completes
